一、生命值类型（派系）

包含：Grineer、Infested、Corpus、Orokin、Sentient、低语者。

派系分配方案：
1. 原版怪物预设派系：
   - Zombie、Drowned、Husk、Zoglin、Zombified Piglin、Zombie Villager、Chicken Jockey、Phantom => Infested
   - Endermite、Enderman、Shulker、Guardian、Elder Guardian => Sentient
   - Skeleton、Wither Skeleton、Stray、Vex、Breeze、Breeze、Ghast、Blaze、Hoglin、Piglin Brute、Piglin、Spider Jockey、Skeleton Horseman 、Slime、Magma Cube=> 低语者
   - 铁傀儡、雪傀儡 => Corpus
   - Warden、Creeper、Charged Creeper、Wither、Ender Dragon => Orokin
   - Witch、Pillager、Evoker、Vindicator、Ravager=> Grineer
   - 玩家 => Orokin
   - 其余的原版中立生物，被动生物，其余的mod怪物默认都为Orokin

2. Mod怪物默认派系：
   - 所有Mod添加的怪物默认分配Orokin派系
   - 可通过配置文件自定义特定Mod怪物的派系归属
   - 支持按命名空间（modid）或具体实体ID设置派系

3. 配置文件结构：
   - 默认派系配置
   - 派系映射配置（支持通配符和精确匹配）
   - Mod级别默认派系配置

三、元素属性及效果说明（含具体数值）

（一）物理元素（冲击、穿刺、切割）触发后效果

• 冲击（物理伤害）：对敌人：（1）蹒跚（后退一格）。对玩家：无效果。  

• 穿刺（虚弱）（物理伤害）：对敌人：（1）伤害输出减少（叠5层，第1层-40%，后续每层-10%，最大-80%，持续6秒）；（2）被暴击率增加（叠5层，每层+5%，最大+25%，直接加算，不适用技能和范围伤害）。对玩家：伤害输出-30%（持续6秒）。  

• 切割（出血）（物理伤害）：通用：每秒造成基础伤害35%的真实伤害（无视护甲），持续6秒（第1-6秒伤害，第0秒无），每层独立，最高10层。注：切割异常伤害不受切割Mod加成，仅提升触发占比；部分无物理伤害武器不受影响。  

（二）单个元素（基础元素：冰冻、电击、火焰、毒素）触发后效果

• 冰冻（基础元素）：对敌人：（1）移速/射速/攻速降低（叠10层，第1层-50%，2-9层每层-5%，9层共-90%；10层冻结3秒并取消护盾充能）；（2）暴击伤害加成（叠10层，第1层+10%，后续+5%，最大+50%；冻结时+100%）。对玩家：移速/射速/攻速降低25%（持续6秒）。  

• 电击（基础元素）：对所有目标：（1）电DOT攻击（每秒50%基础伤害DoT，持续6秒，0-5秒伤害，最高10层）；（2）眩晕（无法移动，持续3秒）。  

• 火焰（基础元素）：对所有目标：（1）减少当前护甲50%（持续6秒）；（2）火焰DoT（每秒50%基础伤害，持续6秒，1-6秒伤害），最高叠10层。

• 毒素（基础元素）：通用：可绕过护盾的毒素DoT（每秒50%基础伤害，持续6秒，1-6秒伤害，最高叠10层）。  

（三）复合元素（爆炸、腐蚀、毒气、磁力、辐射、病毒）触发后效果

• 爆炸（火焰+冰冻）（复合元素）：对敌人：1.5秒后延迟范围伤害（每层30%基础伤害爆炸伤害，叠10层；死亡或满层立即结算，不满层或者没死亡时不造成伤害，范围5格）。  

• 腐蚀（电击+毒素）（复合元素）：对敌人：削甲叠10层（第1层-26%，后续每层-6%，最大-80%，持续8秒）。对玩家：削甲-26%（持续8秒）。  

• 毒气（火焰+毒素）（复合元素）：通用：（1）AoE毒气DoT（每秒50%基础伤害，持续6秒，0-5秒伤害，最高10层）；（2）毒云范围（每层+0.3米，最大+3米至6米）。注：原始目标死亡后毒云仍存。  

• 磁力（冰冻+电击）（复合元素）：对敌人：（1）护盾伤害叠10层（第1层+100%，后续+25%，最大+325%，持续6秒）；（2）护盾再生失效（6秒）；（3）破盾后每层3%护盾电击伤害。对玩家：（1）护盾伤害+100%（3秒）；（2）护盾再生失效（6秒）。  

• 辐射（火焰+电击）（复合元素）：对敌人：（1）敌我不分攻击友军（12秒）；（2）受到辐射伤害叠10层（每层+50%，最大+550%，持续12秒）。  

• 病毒（冰冻+毒素）（复合元素）：对敌人：受到生命值伤害叠10层（第1层+100%，后续+25%，最大+325%，持续6秒）。对玩家：同效果（对目标生命值伤害+100%起，叠层同上）。  
 

四、伤害计算公式

（一）武器面板伤害值

公式：武器面板伤害值 =（冲击 + 穿刺 + 切割）+（元素）。  
元素组合规则：  
• MOD配置界面左上角优先组合，右下角最后组合；固有元素在上述完成后组合。  

• 相同元素MOD以最先使用的决定次序。  

• 本身含复合元素伤害的武器或装复合元素MOD（如毒气铅弹），保留复合元素，可组合特殊组合（如病毒+腐蚀）。  

（二）实际伤害计算
核心逻辑：实际伤害（ID）=基础伤害（BD）× 伤害DM）。

无护甲单位（或真实伤害）：

公式：ID = BD × DM
（DM=1+HM）。

说明：HM为生命克制修正系数（如冲击对Grineer+50%→HM=0.5）；真实伤害（如切割DoT、毒素DoT）无视护甲，用此公式。

有护甲单位：

公式：AM = 0.9 × √(AR/2700)；DM =（1+HM）×暴击伤害×（1-AM）；ID = BD × DM。
说明：AM为护甲减免修正系数（AR为目标最终护甲值，上限2700）；HM为生命克制修正系数。

暴击倍率（暴击伤害） =1 + 暴击等级 × (武器总暴击伤害 − 1)
武器总暴击伤害 = 武器基础暴击伤害 × (1 +暴击伤害增益)
暴击等级判断：n% < 暴击几率 ≤ (n% + 100%)
概率达到的暴击等级：	(n% + 100%) ÷ 100
保底达到的暴击等级：   (n% + 100%) ÷ 100-1



护甲计算公式

（一）护甲系数

•护甲系数=1+0.4×(当前怪物等级-基础等级)^0.75。  

（二）当前护甲值

公式：当前护甲值(AR)=基础护甲×护甲系数。  
• 护甲上限：2700（减免90%伤害）；生效条件：敌人生命条黄色时护甲生效（伤害减免）。 


（三）周期伤害（DoT）单次伤害

公式：周期伤害=武器最终基础伤害×(1+元素Mod伤害)×DoT倍率×(1+派系伤害倍率)。  
• 武器最终基础伤害=基础伤害算完基础Mod后×暴击倍率×(1+派系倍率)×其他加成（怨怒护甲、分子填充、弱点加成等）。  

• DoT倍率：切割0.35，毒素/电击/火焰/毒气0.5。  

• 元素DoT加成：切割、毒气仅受基础Mod影响；毒素/电击/火焰受对应元素Mod+基础Mod影响（赤毒/信条加成不算元素加成）。  

 

六、血量相关说明

• 无护甲单位伤害：直接按"无护甲单位实际伤害"公式计算（涉及HM克制系数）。  

• 有护甲单位伤害：按"有护甲单位实际伤害"公式计算（涉及AM护甲减免）。  

• 血量显示：黄色血条=护甲生效（伤害减免）；基础护甲在资料库/信息盒显示，当前护甲与等级相关（见护甲公式）。
