仓鼠核心 - 深度战斗增强系统

仓鼠核心是一个受星际战甲启发的Minecraft战斗增强模组，为游戏带来完整的元素属性和派系克制系统，彻底改变传统的战斗体验。

核心特色：

六大派系系统
Grineer、Infested、Corpus、Orokin、Sentient、低语者，每个派系都有独特的克制关系。模组智能为原版和Mod怪物分配派系，每种武器都对应特定的克制效果，让战斗更具策略性。

九大元素体系与复合系统
物理元素：冲击造成蹒跚后退，穿刺削弱敌人输出，切割造成持续流血
基础元素：冰冻减速冻结，电击眩晕麻痹，火焰削减护甲，毒素穿透护盾
复合元素：本模组的最大特色，通过不同基础元素组合产生强力效果
- 爆炸(火焰+冰冻)：延迟范围爆炸，清理群体敌人
- 腐蚀(电击+毒素)：强力削甲效果，最高削减80%护甲
- 毒气(火焰+毒素)：制造毒云区域，持续伤害范围内敌人
- 磁力(冰冻+电击)：克制护盾单位，大幅提升护盾伤害
- 辐射(火焰+电击)：让敌人敌我不分，制造战术混乱
- 病毒(冰冻+毒素)：大幅提升对生命值的伤害

深度状态效果系统
每种元素都具备独特的触发机制和叠加效果。冰冻可逐步冻结敌人，10层时冻结3秒并暴击伤害翻倍；腐蚀能大幅削减护甲；辐射让敌人敌我不分，制造战术混乱。状态效果可叠加多层，为战斗增添丰富策略选择。

元素组合策略
模组的精髓在于元素组合。玩家可以通过MOD配置自由组合元素，左上角优先组合，右下角最后组合。比如先装备火焰MOD再装备冰冻MOD会形成爆炸元素，而反过来则可能形成其他复合元素。相同元素的组合顺序也影响最终效果，这为配装提供了极大深度。本身含复合元素的武器还能与其他元素组合特殊效果。

实时战斗信息反馈
攻击时即时显示：敌人等级、派系属性、护甲值、武器属性、元素构成、触发效果等详细信息。帮助玩家了解战斗细节，优化配装策略。从基础伤害到最终伤害的完整计算过程一目了然。

智能兼容性设计
完美支持原版和各类生物Mod，通过配置文件灵活定义派系规则。武器Mod全面兼容，元素效果自动适配，无需额外配置。支持按命名空间、实体ID精确配置怪物派系。

战术深度体现
每一场战斗都需要考虑：武器属性选择对应克制派系，元素组合追求最佳触发效果，装备搭配平衡输出与生存。刷怪效率取决于对系统的理解深度，Boss挑战更考验配装策略。不同的敌人需要不同的元素配装，增加了游戏的可玩性和挑战性。

适合追求硬核战斗体验、喜欢数值计算和策略搭配的玩家。为Minecraft带来接近动作游戏的战斗深度，重新定义战斗的乐趣和挑战。
