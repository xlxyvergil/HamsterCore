# 词缀命令使用说明

## 概述

本模组添加了一个 `/hamsteraffix` 命令，用于管理玩家主手物品上的词缀。

## 权限要求

执行此命令需要权限等级2（默认为管理员级别）。

## 命令结构

```
/hamsteraffix <add|remove|list> [参数...]
```

### 1. 添加词缀

**命令格式：**
```
/hamsteraffix add <玩家> <元素类型> <数值>
```

**参数说明：**
- `<玩家>`: 目标玩家名或选择器
- `<元素类型>`: 要添加的元素类型（支持Tab自动补全）
- `<数值>`: 词缀的数值（可以是小数）

**示例：**
```
/hamsteraffix add Steve heat 10.5
/hamsteraffix add @p gas 5.0
```

### 2. 删除词缀

**命令格式：**
```
/hamsteraffix remove <玩家> <元素类型>
```

**参数说明：**
- `<玩家>`: 目标玩家名或选择器
- `<元素类型>`: 要删除的词缀元素类型（支持Tab自动补全，显示当前物品上的词缀类型）

**示例：**
```
/hamsteraffix remove Steve heat
/hamsteraffix remove @p gas
```

### 3. 列出词缀

**命令格式：**
```
/hamsteraffix list <玩家>
```

**参数说明：**
- `<玩家>`: 目标玩家名或选择器

**示例：**
```
/hamsteraffix list Steve
/hamsteraffix list @p
```

## 元素类型列表

### 基础元素
- `cold` - 冰冻
- `electricity` - 电击
- `heat` - 火焰
- `toxin` - 毒素

### 复合元素
- `blast` - 爆炸 (火焰+冰冻)
- `corrosive` - 腐蚀 (电击+毒素)
- `gas` - 毒气 (火焰+毒素)
- `magnetic` - 磁力 (冰冻+电击)
- `radiation` - 辐射 (火焰+电击)
- `viral` - 病毒 (冰冻+毒素)

### 物理元素
- `impact` - 冲击
- `puncture` - 穿刺
- `slash` - 切割

### 特殊属性
- `critical_chance` - 暴击率
- `critical_damage` - 暴击伤害
- `trigger_chance` - 触发率

### 派系元素
- `grineer` - Grineer派系
- `infested` - Infested派系
- `corpus` - Corpus派系
- `orokin` - Orokin派系
- `sentient` - Sentient派系
- `murmur` - Murmur派系

## 注意事项

1. 词缀只会添加到玩家主手的物品上
2. 如果玩家主手为空或物品不支持词缀，命令会返回错误
3. 添加词缀会自动触发Basic层管理规则：
   - 如果是该元素类型第一次添加到物品上，会添加到Basic层
   - 删除词缀时，如果该类型的词缀完全不存在了，会从Basic层移除
4. 所有词缀操作都会自动保存到物品的NBT中

## 错误处理

如果命令执行失败，系统会显示相应的错误信息：
- 玩家主手上没有物品
- 未知的元素类型
- 物品没有词缀数据
- 物品上没有找到指定类型的词缀
- 权限不足
